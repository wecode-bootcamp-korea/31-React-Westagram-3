## Checkout 11AM
퇴실시간이 11시인 제주도의 고급 숙박 예약 사이트
국내 숙박 예약 사이트 스테이폴리오 를 모티브
<br>
### 📌프로젝트 선정 이유
전체적으로 군더더기 없는 깔끔하고 모던 한 UI
소셜 로그인 API, 지도 API를 써 볼 수 있는 사이트
필터와 검색 시스템이 많아, 연습을 해 볼 수 있고, 백엔드 모델링도 할 것이 많아 전체적인 학습을 하기에 좋음
<br>
